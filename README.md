# Reading Archive (Nuxt)

독서 히스토리를 아카이브하는 Nuxt 웹페이지입니다.

## 프로젝트 구조

```txt
app/
  app.vue
  assets/css/main.css
  components/BookCard.vue
  composables/useReadingArchive.ts
  layouts/default.vue
  pages/index.vue
  types/book.ts
nuxt.config.ts
```

## 실행 방법

```bash
npm install
npm run dev
```

개발 서버: `http://localhost:3000`
